Object Ratio Visualization

About This Tool

This visualization tool displays images from the WikiArt dataset with detected objects, organized by the relative size (ratio) of each object class within the images.

Data Collection Process

The data was generated through the following steps:

  1. Object Detection: We used a pre-trained object detection model (YOLOv5) to analyze all images in the WikiArt dataset.
  2. Ratio Calculation: For each detected object, we calculated its relative size as a percentage of the total image area.
  3. Bucketing: Objects were grouped into 100 buckets based on their size ratios (from 0% to 100% of image area).
  4. Organization: Images were then organized by object class and ratio bucket for visualization.

How to Use This Tool

  • Select a Class: Choose an object class from the dropdown to view images containing that object.
  • Navigate Buckets: Use the up/down buttons (or W/S keys) to move between different size ratio buckets.
  • Browse Images: Use the left/right buttons (or A/D keys) to view different images within the current bucket.
  • Adjust Display: Modify the bounding box opacity and thickness using the sliders.

Understanding the Visualization

Each image shows the original artwork with detected objects highlighted:

  • Bounding Boxes: Colored rectangles around detected objects.
  • Labels: Class name and detection confidence score (0-1).
  • Bucket Range: Shows the percentage range of object sizes in the current bucket (e.g., 10-11% means objects take up 10-11% of the image area).

Technical Details

The visualization is built with:

  • Vanilla JavaScript for the interactive components
  • HTML5 Canvas for image rendering and bounding box display
  • Precomputed JSON data files containing detection results
  • Responsive design that works on desktop and mobile devices
