About This Project
This explorer uses facial emotion detection to analyze artworks. Each artwork is placed on a grid based on the detected emotions.
How to Use
Click on the emotion grid or use the sliders to select an emotion zone. A random artwork from that zone will be displayed with emotion percentages.
